# Crazy Css

The craziest css library to create UI components

## Installation

Install with npm

```shell
npm install crazy
```

Or use a CDN

_Production ready_

```html
<link src="unpkg.com/crazy@latest/css/crazy.min.css" />
```

_Development(not recommended)_

```html
<link src="unpkg.com/crazy@latest/css/crazy.css" />
```
